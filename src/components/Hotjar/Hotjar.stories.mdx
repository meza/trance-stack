# Hotjar

We use [Hotjar](https://www.hotjar.com/) to record user behavior and feedback on our website.
Hotjar is a tool that we use to better understand our usersâ€™ needs and experience.

## Usage:

### Parameters

- `hotjarId`: The ID of the Hotjar project
- `visitorId`: The arbitrary ID of the visitor (userId, email, etc.)

### Example

```jsx
import { Hotjar } from '~/components/Hotjar';

<Hotjar hotjarId={'12345'} visitorId={'abcdef'}/>
```
